Projede Azure Key Vault ve Azure Blob Storage entegrasyonu olup yolladığınız resimleri önceeden key vaulta tanımlı blob storaglerinize dinamik bir şekilde 
istediğiniz kalitede sıkıştırıp yüklemektedir
